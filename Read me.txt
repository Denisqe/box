Приложение для автоматического расчета пульсовых параметров
Для работы требуется bluetooth адаптер на устройстве и веб-браузер последней версии с поддержкой Web Bluetooth API (на старых системах может не работать)
Проверялось на Windows 10, на браузерах Яндекс, Google
Пульсометр Polar h10
На ФПГ пульсометрах замечена неадекватная реакция программы, требует отладки

stable version.html - стабильная рабочая версия, 2 ступени нагрузки (при необходимости можно просто пропустить вторую ступень, зафиксировав расчеты первой)
