Приложение для автоматического расчета пульсовых параметров
Для работы требуется bluetooth адаптер на устройстве и веб-браузер последней версии с поддержкой Web Bluetooth API (на старых системах может не работать)
Проверялось на Windows 10, на браузерах Яндекс, Google

1) Last version.html - последняя версия, адаптирована под одну ступень нагрузки, исправлен интерфейс пользователя, 
применено округление параметров - до конца не отлажено, могу быть неточности

2) stable version.html - стабильная рабочая версия, 2 ступени нагрузки (при необходимости можно просто пропустить вторую ступень, зафиксировав расчеты первой)
